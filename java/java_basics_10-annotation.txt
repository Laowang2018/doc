注释：
    单行注释 //
    多行注释 /* */
    文档注释 /** */

注解：
    写法
        @XXX[(一些信息)]
    注解放置在哪里
        类的上面 属性的上面 方法上面 构造方法上面 参数前面
    注解的作用
        用来充当注释的作用（仅仅是一个文字说明） @Deprecated
        用来做代码的检测（验证） @Override
        可以携带一些信息（内容）
    java中有一些写好的注解供我们使用
        @Deprecated 用来说明方法是废弃的
        @Override 用来做代码检测的 检测方法是否是一个重写方法
